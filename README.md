# LaborPlagScan
Labor Plagiat Scanner für Prog HS-Harz

## Installation:

Benötigt Python 3.11 (Add to PATH aktivieren)



### Anleitung:
1. run.bat (Win) bzw. run.sh (Linux) starten
2. Vollständigen Abgaben Ordner aus StudIP herrunterladen
3. ZIP auswählen (Abgaben-Ordner)
4. PlagScan starten
5. Ergebnisse angucken


### Filter:
1. filter.txt einfach öffnen und Liste anpassen
2. Filter werden automatisch angewendet
3. Syntax:
3.1. Inhalte immer in Anführungszeichen ''
3.2. Varialben Namen (alles was keine Syntax ist) mit x ersetzen
3.3. Für Verwendung von REGEX: 'Regex:[A-Z]*' (Beispiel)
